写文章相关技巧总结，根据实际经验不定期更新。 Post文章管理在hexo生成博客中，所有的文章储存在_posts文件夹里，如果将写好的文章全部放到一个文件夹，随着文章的增多，管理会显得很混乱，于是，我们需要一个好的文章管理方案。 我们认识到： 在文件路径中多一个文件夹不过是在生成博客文章的名字又长了一点，从http://xx/xx.md增长到http://xx/文件夹/xx.md 分类管理，同时避免尽可能多的子类嵌套 方便编辑 所以，有： 在_posts文件夹下可建立多个子文件夹，文件夹可嵌套 根据个人知识体系建立子文件夹，追求扁平化、专题式内容分类 使用专业软件辅助书写和管理文章，如obsidian、typora、vs code 资源链接与参考
